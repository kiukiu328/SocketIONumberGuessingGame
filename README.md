## Socket.IO Number Guessing Game
This is a simple number guessing game using Socket.IO. The server generates a random number between 1 and 100, and the client has to guess it. 

### How to run
1. Clone the repository
2. Run `npm install` to install the dependencies
3. Run `node server.js` or `npm start` to start the server 
4. Open `localhost:6827` in your browser

### How to play
1. Enter your name
2. 1) click `Create Room` for create a room
   2) type the room code then click `Join Room` for join a room
3. Guess the number ^_^
